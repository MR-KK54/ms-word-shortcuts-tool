# MS Word Keyboard Shortcuts Customizer & Import/Export Tool

This is a premium, high-fidelity web application to manage, record, customize, and migrate keyboard shortcuts in Microsoft Word. It provides a visual dashboard for built-in Word commands and allows adding your own custom macros, storing all configuration data securely inside the browser's `LocalStorage`.

## Key Features

1. **Dashboard & Customizer**:
   - Live search, filtering by categories (File, Edit, Formatting, Insert, etc.).
   - Visual keycaps layout representing shortcuts.
   - Edit built-in shortcuts or reset them to defaults.
2. **Add Custom Commands/Macros**:
   - Create records for custom VBA macro shortcuts (`wdKeyCategoryMacro`) or missing built-in commands (`wdKeyCategoryCommand`).
   - Store custom macros and assign shortcuts to save on it.
3. **Interactive Shortcut Recorder**:
   - Record shortcuts directly by pressing keystrokes on your keyboard.
   - Prevents default browser actions so combinations like `Ctrl+S`, `Ctrl+H`, or `Alt+F8` are captured successfully.
4. **VBA Script Installer Generator**:
   - Generates fully commented and executable VBA macro files (`.bas` format).
   - Generates both **Apply Shortcuts** and **Reset Customizations** code blocks.
5. **ListCommands Parser**:
   - Copy-paste the table generated by Word's native `ListCommands` command to automatically sync active word settings back into the web database.
6. **Dark & Light Themes**:
   - Vibrant dark slate mode with blue-violet glows and a light mode option.

## Getting Started

### 1. How to run the Website locally
Since this is a client-side Single Page Application (SPA), you can run it directly in your browser.

To run it with a local HTTP server for full API support (like drag-and-drop or file uploads):
```bash
# In this directory, run a quick server using npx
npx -y http-server
```
Alternatively, double-click the `index.html` file to open it directly in any modern browser.

## MS Word Integration Guide

### How to Import Shortcuts from Word into the Web Tool
1. In Word, press **Alt + F8** (or go to **View** &rarr; **Macros**).
2. Select **Word commands** in the "Macros in" dropdown list.
3. Type `ListCommands` in the name field and click **Run**.
4. Choose **Current keyboard settings** in the dialog box.
5. Select the entire generated table in the new document (press **Ctrl + A**, then **Ctrl + C**).
6. Go to the **Import from Word** tab on the website, paste the text into the paste zone, and click **Parse Paste Content**.
7. Review and import your selected keybinds.

### How to Export Shortcuts from the Web Tool into Word
1. Go to the **Export & VBA** tab on the website.
2. Copy the generated VBA code under the **Apply Shortcuts** tab.
3. Open Microsoft Word, press **Alt + F11** to launch the Visual Basic Editor.
4. Click **Insert** &rarr; **Module** from the top menu.
5. Paste (**Ctrl + V**) the script into the empty module code window.
6. Press **F5** (or click the green arrow icon) to run the installer script.
7. Your shortcuts are immediately active! Close the VBA Editor and enjoy your customized Word workflow.
