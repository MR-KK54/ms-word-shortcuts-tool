Import Existing Shortcuts from Word

Sync your existing Word shortcuts directly into this tool by pasting your current shortcut table.

1 Run the ListCommands macro in MS Word

MS Word has a built-in macro that prints all of your current shortcuts into a new document table.

  1. In Word, go to View → Macros (or press Alt + F8).
  2. In the "Macros in" dropdown list, select Word commands.
  3. In the "Macro name" text field, type ListCommands.
  4. Click Run.
  5. In the dialog that appears, select Current keyboard settings and click OK.
  6. A new document containing a table of your shortcuts is generated!

2 Paste the generated Table here

Select the entire generated document table (or press Ctrl + A in the new document, then Ctrl + C) and paste it into the box below.

How to load the VBA Macro into Microsoft Word

Follow these visual instructions to apply the generated keyboard shortcuts to your copy of MS Word.

1

Copy the VBA Script

Go to the Export & VBA tab, choose the Apply Shortcuts tab, and click Copy VBA. This copies the installation macro script to your clipboard.

2

Open the VBA Editor in Word

Open Microsoft Word. Press Alt + F11 on your keyboard (or go to the Developer tab and click Visual Basic) to open the Microsoft Visual Basic for Applications window.

3

Insert a New Code Module

In the VBA Editor window, click Insert from the top menu, then select Module. A new code editing window will open.

4

Paste & Run the Macro

Press Ctrl + V to paste the VBA script into the empty module window. Then, press F5 (or click the green Run Sub/UserForm arrow icon on the toolbar) to execute the code.

The script will immediately run, configure all key combinations globally, and save them to your Normal.dotm template!
5

Close and Save Word

You can now close the VBA Editor. In Word, your keyboard shortcuts are active immediately! Word will automatically prompt you to save your global settings when closing Word. Make sure to click Save if prompted.
